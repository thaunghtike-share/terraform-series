# Week 6 Day 3 — RBAC & Permission Design

## Learning Objective

Students will understand **RBAC & Permission Design** and apply it in a real enterprise Terraform platform.

---

## Topics

- Least privilege
- Backend access
- Scope strategy

---

## Real Hands-on Tasks

- Assign Reader
- Assign Storage Blob Data Contributor
- Test backend access

---

## Lab Instructions

1. Open the week-6 lab folder.
2. Review the weekly `README.md`.
3. Create or update the required Terraform files.
4. Run Terraform commands safely.
5. Validate the result.
6. Commit the completed work.

---

## Recommended Commands

```bash
terraform fmt -recursive
terraform init
terraform validate
terraform plan
```

For apply sessions:

```bash
terraform apply
```

---

## Student Deliverables

- Working Terraform code
- Clean folder structure
- Updated README notes
- Successful Terraform plan output
- Git commit for the completed day

---

## Notes for Instructor

Focus on real-world explanation.

Do not teach this as syntax only.

Explain why this pattern is used in enterprise infrastructure teams.
