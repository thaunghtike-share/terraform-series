# Advanced Terraform & Platform Engineering Bootcamp

[![Terraform](https://img.shields.io/badge/terraform-%235835CC.svg?style=for-the-badge&logo=terraform&logoColor=white)](https://www.terraform.io)

This repository contains the full course outline and weekly lab guide for the **Advanced Terraform & Platform Engineering Bootcamp**.

This course focuses only on:

- Terraform
- Azure Infrastructure
- Remote State
- Terraform Modules
- GitHub Actions CI/CD
- Azure OIDC
- Terraform Cloud
- terraform-docs
- Terragrunt
- Atmos
- Platform Engineering Workflow

No Docker.  
No Kubernetes.  
No GitOps.

## Course Outcome

By the end of this course, students will build a production-style Terraform platform with:

- Multi-layer infrastructure architecture
- Azure Blob remote state backend
- Reusable Terraform modules
- GitHub Actions CI/CD
- Azure OIDC secretless authentication
- Terraform Cloud workflow
- terraform-docs automation
- Terragrunt structure
- Atmos platform orchestration
- Golden path infrastructure template
