# Week 1 Day 2

## Learning Objective

Understand and implement real enterprise Terraform workflow.

---

## Topics

- Terraform enterprise concepts
- Production repository structure
- Infrastructure automation
- Platform engineering workflow

---

## Real Hands-on Tasks

- Build real infrastructure
- Apply enterprise standards
- Test Terraform workflow
- Validate infrastructure changes

---

## Lab Notes

Students should:

1. Understand the architecture
2. Implement the Terraform code
3. Validate infrastructure changes
4. Troubleshoot issues
5. Commit changes into Git repository

---

## Recommended Commands

```bash
terraform fmt -recursive
terraform init
terraform validate
terraform plan
terraform apply
```

---

## Expected Deliverables

- Working Terraform code
- Clean repository structure
- Successful Terraform plan
- Git commits for completed work
